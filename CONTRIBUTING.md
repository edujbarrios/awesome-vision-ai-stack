# Contributing

Thanks for helping improve **Awesome Vision AI Stack**.

## What belongs here

Add resources that are:

- practical and useful for builders
- relevant to VLMs, visual agents, document AI, UI understanding, grounding, robotics, or multimodal evaluation
- maintained or historically important
- reasonably discoverable and well documented

## What does not belong here

Please avoid adding:

- low-quality clones
- pure paper spam with no practical relevance
- dead repos with no historical importance
- generic LLM tools that do not clearly help vision workflows

## Pull request format

For each addition, try to include:

- project name
- link
- one-line description
- the section where it fits best

Example:

```md
- [Project Name](https://github.com/org/repo) - Clear one-line description.
```

## Recommended criteria

A resource is especially valuable if it helps with at least one of these:

- local deployment
- document understanding
- screenshot and UI understanding
- vision agents and action
- robotics or grounding
- evaluation and benchmarking
- production serving

## Style

- Keep descriptions short.
- Use sentence case.
- Prefer direct repo or official links.
- Avoid hype language.
- Keep the list builder-first.

## Big idea

This repo is trying to be the practical counterpart to research-heavy multimodal lists.

If your addition helps someone ship a vision product faster, it probably belongs here.
